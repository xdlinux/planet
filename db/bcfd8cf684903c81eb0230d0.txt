<h1><a id="content-现象" href="https://blog.lsy223622.com/archives/59/#content-%E7%8E%B0%E8%B1%A1" class="heading-permalink" aria-hidden="true" title="Permalink"></a>现象</h1>
<ul>
<li>打开 Windows 资源管理器。</li>
<li>右击想要共享的文件夹，选择 <code>属性</code>。</li>
<li>弹出的窗口没有 <code>共享</code> 标签页。</li>
</ul>
<h1><a id="content-解决方法" href="https://blog.lsy223622.com/archives/59/#content-%E8%A7%A3%E5%86%B3%E6%96%B9%E6%B3%95" class="heading-permalink" aria-hidden="true" title="Permalink"></a>解决方法</h1>
<ol>
<li>
<p>按下 <code>Win + R</code> 组合键，打开 <code>运行</code> 窗口，或打开左下角的搜索框。</p>
</li>
<li>
<p>输入 <code>regedit</code> 并回车，打开 <code>注册表编辑器</code>。</p>
</li>
<li>
<p>前往 <code>HKEY_CLASSES_ROOT\Directory\shellex\PropertySheetHandlers\Sharing</code></p>
<ul>
<li>如果找不到上方路径，请在 <code>PropertySheetHandlers</code> 文件夹里，创建名为 <code>Sharing</code> 的文件夹。</li>
</ul>
</li>
<li>
<p>双击 <code>(默认)</code> REG_SZ 机码，把数值改为 <code>{f81e9010-6ea4-11ce-a7ff-00aa003ca9f6}</code> 并确定。</p>
</li>
<li>
<p>重启资源管理器，此时右击文件夹，选择 <code>属性</code>，就会看到 <code>共享</code> 标签页了。</p>
</li>
</ol>

<h1><a id="content-其他解决方法" href="https://blog.lsy223622.com/archives/59/#content-%E5%85%B6%E4%BB%96%E8%A7%A3%E5%86%B3%E6%96%B9%E6%B3%95" class="heading-permalink" aria-hidden="true" title="Permalink"></a>其他解决方法</h1>
<blockquote>
<p>我的电脑用上面的方法解决了，但考虑到不同的电脑可能会有不同的问题，这里列出了一些其他的解决方法。</p>
</blockquote>
<ul>
<li>
<p>启用 <code>服务</code> 中的 <code>Server</code> 和 <code>Computer Browser</code> 服务。</p>
<ol>
<li>按下 <code>Win + R</code> 组合键，打开 <code>运行</code> 窗口，或打开左下角的搜索框。</li>
<li>输入 <code>services.msc</code> 并回车，打开 <code>服务</code>。</li>
<li>在打开的服务列表中，找到 <code>Server</code> 服务，双击打开。</li>
<li>在弹出的窗口中，将 <code>启动类型</code> 设置为 <code>自动</code>，然后点击右下角的 <code>应用</code> 按钮。</li>
<li>在服务列表中，继续找到 <code>Computer Browser</code> 服务，同样双击打开。</li>
<li>在弹出的窗口中，同样将 <code>启动类型</code> 设置为 <code>自动</code>，最后点击右下角的 <code>应用</code> 按钮。</li>
<li>重启电脑，再次右击文件夹，选择 <code>属性</code>，就会看到 <code>共享</code> 标签页了。</li>
</ol>
</li>
<li>
<p>开启 <code>SMB 1.0/CIFS 文件共享支持</code></p>
<ol>
<li>打开左下角的搜索框，输入 <code>控制面板</code> 并回车，打开 <code>控制面板</code>。</li>
<li>在 <code>控制面板</code> 中，选择 <code>程序</code> -&gt; <code>程序和功能</code>。</li>
<li>在打开的窗口中，点击左侧的 <code>启用或关闭 Windows 功能</code>。</li>
<li>在弹出的窗口中，找到 <code>SMB 1.0/CIFS 文件共享支持</code>，点击左边的 + 号展开。</li>
<li>勾选 <code>SMB 1.0/CIFS 服务器</code> 和 <code>SMB 1.0/CIFS 客户端</code>，然后点击右下角的 <code>确定</code> 按钮。</li>
<li>重启电脑，再次右击文件夹，选择 <code>属性</code>，就会看到 <code>共享</code> 标签页了。</li>
</ol>
</li>
<li>
<p>在 <code>文件夹选项</code> 中勾选 <code>使用共享向导</code></p>
<ol>
<li>
<p>打开 Windows 资源管理器。</p>
</li>
<li>
<p>打开文件夹选项。</p>
<ul>
<li>在 Windows 11 中，点击中间上方的 <code>…</code>，然后点击 <code>选项</code>。</li>
<li>在 Windows 10 中，点击左上角的 <code>查看</code>，然后点击右上角的 <code>选项</code>。</li>
</ul>
</li>
<li>
<p>在打开的窗口中，选择 <code>查看</code> 标签页。</p>
</li>
<li>
<p>在 <code>高级设置</code> 中，找到 <code>使用共享向导(推荐)</code>，勾选后点击右下角的 <code>确定</code> 按钮。</p>
</li>
<li>
<p>重启资源管理器，再次右击文件夹，选择 <code>属性</code>，就会看到 <code>共享</code> 标签页了。</p>
</li>
</ol>
</li>
</ul>
