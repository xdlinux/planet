<h1><a id="content-西安有什么好吃的地方" href="https://blog.lsy223622.com/archives/41/#content-%E8%A5%BF%E5%AE%89%E6%9C%89%E4%BB%80%E4%B9%88%E5%A5%BD%E5%90%83%E7%9A%84%E5%9C%B0%E6%96%B9" class="heading-permalink" aria-hidden="true" title="Permalink"></a>西安有什么好吃的地方？</h1>

<h2><a id="content-好吃多吃" href="https://blog.lsy223622.com/archives/41/#content-%E5%A5%BD%E5%90%83%E5%A4%9A%E5%90%83" class="heading-permalink" aria-hidden="true" title="Permalink"></a>好吃多吃</h2>
<details>
<summary>济州家韩式自助烤肉（性价比高的自助烤肉选这家）</summary>
<ul>
<li>大众点评链接：<a rel="noopener noreferrer" href="https://m.dianping.com/shop/G8OTE7Fl4jhtOkjg">【济州家韩式自助烤肉(丈八三路店)】</a>
</li>
<li>价格：人均六七十</li>
<li>评价：
<ul>
<li>店在韩国风情街，去吃过好几次了，性价比真的很高。</li>
<li>肉大多是整肉，腌制的占比也不是特别大。</li>
<li>肉以外的东西比较少，但我的目的就是吃肉，问题不大。</li>
<li>以前肉的品种不算很多，但是今年去的时候又新增了一个冰柜放了几种看起来不错的肉，爽吃。</li>
<li>服务很积极，几个服务员一直在旁边站着，还挺可怜的。</li>
<li>去的时候店里总是没几桌人，甚至有时候就我们一桌，不知道这几年它有没有真的赚到钱。</li>
<li>每次走的时候都会打一个冰淇淋边走边吃，养成习惯了。</li>
</ul>
</li>
</ul>
</details>
<details>
<summary>谷町·烤肉自助（档次高一些的自助烤肉选这家）</summary>
<ul>
<li>大众点评链接：<a rel="noopener noreferrer" href="https://m.dianping.com/shop/E8vFPX9RqX055C08">【谷町·烤肉自助(朱雀门店)】</a>
</li>
<li>价格：人均一百多点</li>
<li>评价：
<ul>
<li>城墙里面和长安万科都有一家，我都吃过，听别人说城墙里面那家更好。</li>
<li>推荐 138 那一档的，团购优惠能做到 110+，这一档一般都是相对来说比较划算的。</li>
<li>它的肉是小程序点了上来的，看起来比济州家的高级一些，还有一些能自取的肉和别的吃的。</li>
<li>说到烤肉的方法，那些比较薄的肉原则基本上就是烤第一面的时候看到上面开始渗水就可以翻面了，翻面之后看大概差不多就可以吃了，不用翻好几次。</li>
<li>这家人也比济州家多不少，饭点基本上是坐满的，想要不排队也许要早点去。</li>
</ul>
</li>
</ul>
</details>
<details>
<summary>江里江外美蛙自助火锅（爽吃牛蛙）</summary>
<ul>
<li>大众点评链接：<a rel="noopener noreferrer" href="https://m.dianping.com/shop/l7lhppAh7cf41Yld">【江里江外美蛙自助火锅(高新万达店)】</a>
</li>
<li>价格：人均一百不到一点点</li>
<li>评价：
<ul>
<li>连锁店，西安好几家，我只去过高新万达那家，那里人没有特别多而且地铁六号线直达（准确地来说木塔寺站下地铁还要走几百米）。</li>
<li>价格是每个人 73/78 再加一桌一个锅底 39，所以两个人吃的话就是人均一百不到一点点。</li>
<li>感觉西安卖牛蛙的地方好少，其中很多也只是把牛蛙作为普通的一个配菜。</li>
<li>这家主打就是自助的牛蛙，跟服务员说上几只，过会儿就会端上来。</li>
<li>牛蛙看起来是现杀的，至少一盘放在上面的几只撒了盐，腿还在蹬。</li>
<li>下面的几只一般会小一点，但毕竟牛蛙确实是有大有小的，小的也一样好吃，也无所谓。</li>
<li>锅底可以选一种也可以选两种，辣锅用的是火锅底料，里面的辣椒和花椒真的好多好多，就算点的是双拼锅底，微辣那部分用漏勺感觉也能捞三满盘辣椒和花椒，辣得我受不了。</li>
<li>于是第二次和也不怎么能吃辣的舍友去吃了纯不辣的锅底，基本上就是清汤。</li>
<li>牛蛙涮 10～12 分钟出来很鲜嫩，超时就老了，所以吃得多的话可以在上一波熟了的时候就叫服务员上下一波，这样效率比较高。</li>
<li>牛蛙以外的涮菜都是自取，品种很多，蔬菜、菌菇、丸子/淀粉类、科技肉都有，甚至还有好几种水产品和血、脑之类的东西。</li>
<li>炸物、水果、凉拌菜、卤鸭卤鸡腿之类的也都好吃，冰箱里的饮料基本上也是大牌子。</li>
<li>基本上不用排队，周末的中午也没坐满，随时都能去。</li>
</ul>
</li>
</ul>
</details>
<details>
<summary>佛伦萨·古典火炉披萨（感觉比必某客好吃）</summary>
<ul>
<li>大众点评链接：<a rel="noopener noreferrer" href="https://m.dianping.com/shop/l15wYqJN9Qcad2ye">【佛伦萨·古典火炉披萨(高新店)】</a>
</li>
<li>价格：人均几十到一百左右</li>
<li>评价：
<ul>
<li>在西安也是连锁店，这家离学校相对来说近一点，只去过这家。</li>
<li>整体算是正常价位，做披萨和其他西餐水平挺不错的。</li>
<li>烤披萨的大炉子挺好看，不过里面看起来是一圈喷气的火焰，要是柴火或者炭火就更有感觉了（不过不知道合不合适）。</li>
<li>披萨和意面其实没有太多特别的，只要面和酱不出大问题就算能吃了，好吃无非是饼别太厚、奶酪够多、放的料够实诚，这家做到了所以喜欢吃。</li>
</ul>
</li>
</ul>
</details>
<details>
<summary>acquolina·阿果里呐意大利比萨店（披萨性价比很高）</summary>
<ul>
<li>大众点评链接：<a rel="noopener noreferrer" href="https://m.dianping.com/shop/iU4E8I0xf9UKA611">【acquolina·阿果里呐意大利比萨店(紫薇尚层店)】</a>
</li>
<li>价格：人均几十</li>
<li>评价：
<ul>
<li>意大利人开的一家小店，老板看起来确实是外国人，中文会但是会得不多。</li>
<li>前两年朋友发现然后约去吃，起初印象最深的是老板在外卖平台上很爱回复“永别了”（估计是机翻吧）。</li>
<li>店里的披萨都是 16 寸的，一份是 1/4 个，相当于一整个 8 寸的面积。</li>
<li>一份披萨价格都在 20 上下，可以说是在披萨里面很划算了。</li>
<li>千层面和芝士蛋糕很好吃，就是有点贵。</li>
<li>汤我没喝明白，可能这就是意大利风味吧。</li>
<li>今天再看大众点评，这家小店居然上了年度必吃榜，提早发现的我感觉还是挺荣幸的。</li>
</ul>
</li>
</ul>
</details>

<details>
<summary>魏客Channel We Coffee（好吃不贵的西餐厅~）</summary>
<ul>
<li>大众点评链接：<a rel="noopener noreferrer" href="https://m.dianping.com/shop/k5Ph1sUiuz2bI0QL">【魏客Channel We Coffee】</a>
</li>
<li>价格：人均几十</li>
<li>评价：
<ul>
<li>好像西安就一家，但应该也是魏家的一个牌子。</li>
<li>店在绿地缤纷荟/益田假日里的附近，地铁丈八一路站出来就是。</li>
<li>和朋友一起去吃过一次，首先是感觉点的几样东西都蛮好吃的。</li>
<li>比较有印象的是它的那个双椒鸡乌冬面，味道和口感都真的有点惊艳，可以说是融合得特别成功的一个菜。</li>
<li>吃得很饱（我胃口真蛮大的），结账一看，两个人才一百三十块，性价比很不错，远远超过我对西安的西餐厅的预期。</li>
<li>这家店排队的人超级多，之前和舍友晚上七点这样去，门口等满了人，问了说是九点都排不上，到时候厨师都要下班了，于是那次放弃。</li>
<li>所以就是说最好工作日中午去，或者晚上早点去，周末节假日的话自己看着办吧。</li>
</ul>
</li>
</ul>
