<h1><a id="content-现象" href="https://blog.lsy223622.com/archives/4/#content-%E7%8E%B0%E8%B1%A1" class="heading-permalink" aria-hidden="true" title="Permalink"></a>现象</h1>
<ul>
<li>
<p>宝塔面板点击 <code>Docker</code> 菜单，提示 <code>当前未安装 docker 或 docker-compose 未安装</code> 。</p>
</li>
<li>
<p>终端输入 <code>docker -v</code> 能正常看到 Docker 版本信息。</p>
</li>
<li>
<p>重启或重装 Docker 也没有变化。</p>
</li>
</ul>
<h1><a id="content-解决方法" href="https://blog.lsy223622.com/archives/4/#content-%E8%A7%A3%E5%86%B3%E6%96%B9%E6%B3%95" class="heading-permalink" aria-hidden="true" title="Permalink"></a>解决方法</h1>
<ul>
<li>终端输入以下命令：</li>
</ul>
<pre><code class="language-bash">Compose_Download_Url=&#34;http://download.bt.cn/install/src/docker-compose-$(uname -s)-$(uname -m)&#34;
Compose_Path=&#34;/usr/local/bin/docker-compose&#34;
Compose_lin=&#34;/usr/bin/docker-compose&#34;
curl -sSL $Compose_Download_Url -o $Compose_Path
ln -sf $Compose_Path $Compose_lin
</code></pre>
<ul>
<li>
<p>刷新宝塔面板网页。</p>
</li>
<li>
<p>此时应该已经可以正常使用宝塔面板的 Docker 管理页面。</p>
</li>
</ul>
